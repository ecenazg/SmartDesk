# Returns & Damaged Shipments SOP

## Overview
This document outlines the standard procedure for handling returns and damaged shipments
across all departments.

## Reporting Damaged Shipments
Damaged shipments must be reported within 48 hours of delivery.
Employees should photograph the damage before opening the packaging.
All damage reports must be submitted via the Returns Portal with order number and photos attached.

## Return Process
1. Initiate a return request through the Returns Portal
2. Receive a Return Merchandise Authorization (RMA) number within 24 hours
3. Package the item securely with the RMA number clearly visible
4. Drop off at any authorized carrier location
5. Refund or replacement is processed within 5 business days of receiving the item

## Escalation Policy
If the return value exceeds $500, the request requires manager approval before processing.
Contact the Operations team via Slack (#ops-team) for high-value return escalations.
